# Prompts Are Code

I added a few line breaks to a prompt, and execution accuracy visibly improved.

It was a reminder: prompts are code. The only difference is the runtime — a model instead of a compiler. Code values readability, and so do prompts: well-structured instructions simply get misexecuted less.

Most people treat prompts like casual chat — write whatever comes to mind, then complain the AI doesn't listen.

The rigor you put into your prompt is the rigor AI puts into your request.
