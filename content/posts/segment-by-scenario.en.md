# Segment by Scenario, Not by Person

Product people love labeling users: geek, expert, novice.

But the same person is a geek when coding, a novice when facing legal paperwork, and an expert demanding pixel-level control when writing copy.

User segmentation should never cut by person. It cuts by scenario.

That's why a truly universal productivity tool needs three entrances to the same feature: an API for geeks, precise controls for experts, one click with sane defaults for everyone else. This used to be prohibitively expensive to build. AI just made it possible for the first time.
