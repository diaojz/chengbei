# The Bar Didn't Drop. It Moved.

AI didn't lower the bar for building software. It moved the bar.

The hard part used to be execution: memorizing syntax, configuring environments, deciphering errors. AI absorbed all of that. What's hard now is three other things — can you state the problem clearly, can you break the work down, can you tell whether what AI gave you is right.

So with everyone using the same AI, the gap between people isn't shrinking. It's being amplified. Same tool, different leverage.

It's not "anyone can pull it off easily now." It's "anyone gets to start now." Those are very different sentences.
