02

地基层 · 壳与框架

所有东西都站在这两个上面
🏗️
Layer 01 · 地基
窗口 + 插件框架
Ghostty 是那个会发光的窗口本体；窗口里跑的是 zsh，Oh My Zsh 是给 zsh 装插件、管别名的「框架」。
归档关键词
GPU 加速文本配置插件容器
一句话：Ghostty 管「窗口长什么样、怎么分屏」，Oh My Zsh 管「往 zsh 里塞哪些能力」。
这一层装了什么
  • Ghostty——又快又干净的终端，配置就一个文本文件 ~/.config/ghostty/config，存盘即时生效。
  • Oh My Zsh——zsh 的插件框架。后面的 git、z、自动补全、语法高亮全挂在它的 plugins=(...) 里。
  • 注意分工——Oh My Zsh 自带的「主题」我们关掉（ZSH_THEME=""），提示符让位给颜值层的 starship。
03

颜值层 · 提示符 / 字体 / 主题

第一眼的观感，也最容易踩错名字
🎨
Layer 02 · 颜值
好看 + 好读
那行带目录和 git 分支的提示符是 starship 画的；能画出图标，靠的是 Nerd Font；配色由主题决定。
💡
本课提示符统一用 starship：跨 shell 通用、配置就一份 starship.toml，比 p10k 等老方案更省心——那行带目录和 git 分支的提示符就是它画的。
这一层装了什么
  • starship——提示符引擎，自动显示目录 / git 分支 / 语言版本 / 时间。配置在 ~/.config/starship.toml。
  • Nerd Font（JetBrainsMono）——带图标字形的字体，是颜值层地基：没它，eza 图标和分支符号全变方块。
  • 主题——浅色 GitHub Light + 深色 Catppuccin Mocha，跟随系统切换；配 minimum-contrast 防灰字糊。
04

导航层 · 跳目录 / 看文件

终端最累的活，交给它俩
🦆
Layer 03 · 导航
少敲路径
不想一层层 cd？zoxide 记住你去过的目录，敲一半就跳；想像 Finder 一样翻文件？用 yazi（读音「鸭子」）。
归档关键词
智能跳转可视化浏览图片预览
冷知识：yazi 读音像「鸭子」（yā-zi），所以特别好记。
这一层装了什么
  • zoxide——比 cd 聪明，zi 关键词 直接跳历史目录。我还并存了 omz 的 z 和 autojump，挑顺手的用。
  • yazi——终端版文件管理器，方向键浏览、图片预览，按 z 还能调 zoxide 跳转。
  • 关键一步——配 y() 包装函数：退出 yazi 时，shell 自动切到你最后停的目录。
05

现代 CLI 全家桶 · 替换老命令

2026 公认的 Rust 系利器
🧰
Layer 04 · 全家桶
老命令现代化
把上古的 ls / cat / find / grep 全换成带颜色、带图标、更快的现代版。它们也是 yazi 搜索的底层，一起装最省心。
归档关键词
彩色输出默认忽略垃圾目录极快
联动：yazi 里按搜索，背后调的就是这桶里的 fd / ripgrep / fzf。
五件套对照
  • eza 替代 ls——图标 + 颜色 + git 状态 + 树形视图。
  • bat 替代 cat——语法高亮 + 行号 + diff 标记。
  • fd 替代 find——语法人话，自动跳过 .git / node_modules。
  • ripgrep（rg） 替代 grep——文件内容全文搜，极快。
  • fzf——万能模糊搜索，Ctrl+R 翻历史、Ctrl+T 选文件。
06

剩下三层 · 效率 / 监控 / 窗口

打字更稳、看得见用量、一屏多用
LAYER 05 · 效率层打字辅助
autosuggestions 灰字预测整条命令 · syntax-highlighting 命令对绿错红 · git 插件给一堆短别名（gst / gco）。都写进 plugins=(...) 即用。
LAYER 06 · 监控层claude-hud
看 AI 用量：在 Claude Code 底部状态栏显示模型 / 上下文占比 / 5 小时 · 7 天额度恢复时间 / 当前任务。注意它属于 Claude Code 层，不是 zsh 插件。
LAYER 07 · 窗口管理Ghostty 快捷键
无需 tmux：cmd+u 右分屏、cmd+shift+d 下分屏、cmd+shift+f 单屏放大、cmd+t 新标签；cmd+` 从屏幕顶部呼出下拉终端（Quick Terminal）。写进 config 即用。

它们之间什么关系？

谁装在谁里、谁依赖谁

三条主线，一次说清

别把 12 个工具记成 12 件孤立的事——它们其实是三条嵌套关系。

最外层
Ghostty 窗口
（字体 / 主题 / 分屏归它）
→
窗口里跑
zsh + Oh My Zsh
（插件 / 别名挂这里）
→
提示符独立接管
starship
（omz 主题关掉让位）
→
工具自由调用
全家桶 + yazi
（互相调用搜索）
🔗
Nerd Font 是前提：eza 的文件图标、starship 的分支符号，没字体全是方块——所以颜值层先装字体。
🔗
fd / rg / fzf 是 yazi 的底层：yazi 里搜文件、跳目录，背后就是调它们——所以导航层和全家桶要一起装。
🔗
starship ≠ omz 主题：提示符交给 starship，必须把 ZSH_THEME 留空，否则两套抢着画会乱。
🔗
claude-hud 在另一层：它装在 Claude Code 里、不是终端插件，归档时单独放「监控层」别混进 zsh。

一页速记

七层各管一摊，照《配置清单》逐项装，就能 1:1 复刻这块屏幕。

先装什么
Homebrew → Nerd Font → Ghostty，地基和字体先到位，后面才不出方块。
核心五件套
starship · eza · bat · zoxide · fzf，装完终端立刻「换了个人」。
怎么落地 ✅
把《配置清单》里的命令逐条复制给 Claude Code，说「帮我装这个并配好 .zshrc」。
🪟
Windows 同学：本套配置面向 macOS。请改用 Windows Terminal + WSL2，里面同样能装 starship / eza / zoxide 这套全家桶，思路完全一致。
